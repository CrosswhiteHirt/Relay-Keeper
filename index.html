A calm puzzle about controlled flow.

Relaykeeper is not a network simulator. It turns intuitive ideas like congestion, priority, delay, backup paths and rule slots into readable logic puzzles.

Set rules first

Choose a node, place a routing rule, then run the system to see how packets react.

Read clear feedback

Reports explain route changes, timing issues and why a delivery failed.

Refine the machine

Use relays, queues, gates and switches to protect the route with fewer moves.
